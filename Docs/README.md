# Documentation Configuration

Ce fichier permet de configurer le comportement de la documentation.

## Configuration de développement

Pour modifier la documentation localement :

1. Modifier les fichiers `.md` dans ce dossier
2. Le serveur `mkdocs serve` se recharge automatiquement
3. Vérifier les changements sur http://127.0.0.1:8000

## Ajout de nouvelles pages

1. Créer un fichier `.md` dans le dossier approprié
2. Ajouter la référence dans `mkdocs.yml` (racine du projet)
3. La page apparaîtra automatiquement dans la navigation

## Configuration avancée

Voir le fichier `mkdocs.yml` à la racine du projet.
